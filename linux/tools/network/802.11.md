# 802.11

## 規格一覧

* 参考) https://ja.wikipedia.org/wiki/IEEE_802.11
1. 802.11
  * 2.4GHz
  * 2Mbps
2. 802.11a
  * 5GHz
  * 54Mbps
3. 802.11b
  * 2.4GHz
  * 11-22Mbps
4. 802.11g
  * 2.4GHz
  * 54Mbps
5. 802.11n
  * wifi4とも呼ばれる
  * 2.4, 5GHz
  * 65-600Mbps
6. 802.11ac
  * wifi5とも呼ばれる
  * 5GHz
  * 292Mbps-6.9Gbps
    * 実際には市販されているwifi機器ではアンテナ数等の関係で6.9Gbpsでの通信は不可
7. 802.11ax
  * wifi6とも呼ばれる
  * 2.4, 5GHz
  * 9.6Gbps
