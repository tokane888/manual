### Dockerfileから作成

* Dockerfile作成
* 例）
    ```
    FROM jrei/systemd-ubuntu:16.04
    RUN mkdir -p /root/.local/dotfiles
    CMD [ "top" ]
    ```
* ビルド
    * `docker build -t dotfiles:1 .`
        * -t "image名:tag名"形式でdocker image作成
            ```
            REPOSITORY            TAG                 IMAGE ID            CREATED             SIZE
            dotfiles              1                   8439ed093f0d        12 seconds ago      121MB
            ```
